# 🐟 MoyuShuang (魔芋爽)

**MoyuShuang** 是一款基于 Go 语言开发的终端阅读器。它不仅是一个文本查看工具，更是为开发者量身定制的避难所。通过在终端模拟高频滚动的内核日志和系统输出，它能让你在繁忙的办公环境下，神不知鬼不觉地阅读本地文档。

> **“魔芋爽”是一种用于衡量“摸鱼爽度”的离散统计单位。** 
> **经实测，MoyuShuang 系统在生产环境中的部署，能显著提高该指标的平均统计值与峰值响应。**

---

## ✨ 核心特性

* **日志高亮**：日志消息根据严重程度（INFO, DEBUG, ERROR）进行不同颜色高亮显示，增强可读性，即便你根本不会读。
* **BOSS 键支持**：`Space` 键即刻触发 `SUSPENDED` 状态，瞬间隐藏阅读内容。
* **进度自动同步**：程序自动记录每个文件的阅读位置和窗口高度偏好。
* **动态文本折行**：智能计算终端宽度，自动对长文本进行切分，确保阅读体验连续。
* **自定义日志源**：支持自定义日志作为背景，实现 100% 真实环境伪装。
* **交互丰富**：支持搜索 (`/`)、精准跳转 (`G`)、鼠标滚轮操作以及阅读区高度调节。

## 🛠️ 安装与编译
可以直接下载预编译的二进制文件，也可以通过编译源代码来安装。

### 直接下载
在 [Releases](https://github.com/jojenh/moyu/releases) 页面下载适合你操作系统的二进制文件。

### 编译源代码
确保你已安装 Go 环境 ($>= 1.18$)，执行以下命令：

```bash
# 安装必要依赖
go get [github.com/nsf/termbox-go](https://github.com/nsf/termbox-go)
go get [github.com/mattn/go-runewidth](https://github.com/mattn/go-runewidth)

# 编译项目
go build -o MoyuShuang main.go
```

# 🚀 使用指南
## 基本启动
```bash
./MoyuShuang <待摸的鱼.txt>
```
## 高级启动（使用真实日志伪装）
如果你想让界面看起来更像是在处理特定的工作，可以传入一个包含真实日志的文件，就像：
```bash
./MoyuShuang novel.txt my_project_build.log
```
# 🎮 控制指令
## 阅读控制
| 按键 | 动作 |
| --- | --- |
| j / 鼠标左键 | 下一行 |
| k / 鼠标上 | 上一行 |
| + / - | 增加/减少底部阅读区域的高度 |
| Space (空格) | BOSS 模式：立即隐藏内容并显示内核挂起状态 |
## 🔍 导航与搜索
| 按键 | 动作 |
| --- | --- |
| / | 进入搜索模式（输入关键词后回车） |
| n / N | 下一个 / 上一个搜索匹配项 |
| G | 进入行号跳转模式（输入行号后回车） |
| Esc | 退出当前模式 / 取消搜索高亮 |
## ⚙️ 系统指令
| 按键 | 动作 |
| --- | --- |
| h / ? | 显示/隐藏帮助手册 |
| Q | 安全退出并保存当前进度 |
# 📂 进度保存
软件会在用户家目录下生成一个 .moyu_progress.json 文件。它记录了：每个文件的最后阅读行号以及当前窗口的高度偏好。
